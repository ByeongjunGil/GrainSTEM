# Raw 4D-STEM datasets

Place the raw 4D-STEM NBD datasets here, one `.h5` file per ROI, named to match the
`dataset_name` used in the notebooks:

```
data/Gua20_Acq1.h5
data/Gua20_Acq2.h5
data/Gua20_Acq3.h5
data/Gua20_Acq4.h5
data/Gua25_Acq1.h5
data/Gua25_Acq2.h5
data/Gua25_Acq3.h5
data/Gua25_Acq5.h5
```

These files are not tracked by git (see `.gitignore`) because of their size.
Download them from the archive listed in the Data Availability Statement of the paper.
